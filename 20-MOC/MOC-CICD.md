  - lint      # Проверка кода
  - test      # Тесты
  - build     # Сборка
  - security  # Security scan
  - deploy    # Развёртывание

## 📊 Сравнение CI/CD систем

| Система | Плюсы | Минусы |
|---------|-------|--------|
| GitHub Actions | Интеграция с GH, простой YAML | Лимиты на бесплатном тарифе |
| GitLab CI | Всё в одном, мощный | Сложнее в настройке |
| Jenkins | Гибкий, много плагинов | Требует поддержки |
| CircleCI | Быстрый, облачный | Дорогой для команд |

## 🎯 Best Practices

- ✅ Храните секреты в secrets manager
- ✅ Используйте кэширование зависимостей
- ✅ Параллельте независимые задачи
- ✅ Добавляйте уведомления в Slack/Telegram
- ✅ Имейте rollback стратегию
